# Gutix IA — Landing Page

Sitio web oficial de [Gutix IA](https://gutixia.com), agencia de inteligencia artificial.

## Stack

- HTML + CSS + JS vanilla (sin frameworks)
- Deploy en Netlify
- Formulario con Netlify Forms
- Logo SVG vectorial inline

## Estructura

```
├── index.html       # Página principal (todo en un archivo)
├── gracias.html     # Página de confirmación tras enviar formulario
├── netlify.toml     # Configuración de Netlify (headers, caché, redirects)
└── README.md
```

## Deploy

1. Push a GitHub
2. Conectar repo en [netlify.com](https://netlify.com)
3. Publish directory: `.` (raíz)
4. Deploy — listo

## Formulario de contacto

Usa **Netlify Forms**. Los mensajes llegan al dashboard de Netlify.

Para activar notificaciones por correo:
1. Netlify Dashboard → Forms → contacto
2. Settings → Form notifications → Add notification → Email
